# simple-notes-application-300741-300756

This workspace contains the React frontend for a simple notes application.

- Container: frontend_react_notes
- Framework: React (Create React App)
- Port: 3000

Quick start:
1. cd frontend_react_notes
2. npm install
3. npm start
4. Open http://localhost:3000

Features:
- Create, edit, delete notes
- Search and sort
- localStorage persistence (no backend)
- Modern light theme
